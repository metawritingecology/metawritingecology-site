# Public Version Notes

Public pages and publication records are access surfaces for Meta-Writing Ecology.

They help readers locate current public orientation, citation records, fiction surfaces, and platform links. They do not expose the full archive, registry state, private working layers, or unpublished release workflow.

## Boundary Notes

- A publication artifact is not automatically canonical.
- A GitHub or OSF page is an access surface.
- A fiction page is not a model proof.
- A summary is not a source replacement.
- A public page is not the full archive.

## How To Read These Surfaces

Use public pages as orientation and traversal aids.

Use publication records as public citation anchors.

Use fiction pages as entry surfaces for fiction, not as theory proofs or manuscript substitutes.

For broader exclusions, see [Public Boundary](/boundary/). For citation and traversal surfaces, see [Public Record Anchors](/public-records/).
