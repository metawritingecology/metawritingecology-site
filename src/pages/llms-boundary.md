# llms.txt Boundary

The site-level `llms.txt` file is part of Meta-Writing Ecology's AI-readable public boundary layer.

It is not an AI SEO strategy, ranking-optimization mechanism, traffic-growth mechanism, chatbot instruction layer, full-corpus release, or full-system exposure mechanism.

`llms.txt` provides a compact public orientation surface for automated readers. It supports source positioning, summary boundaries, citation visibility, and misreading reduction.

It should be read together with:

- [AI-Readable Knowledge Architecture](/ai-readable-knowledge-architecture/)
- [Interpretation Boundaries](/interpretation-boundaries/)
- [AI Training Boundary](/ai-training-boundary/)
- [Application Boundary](/application-boundary/)

Minimal distinctions:

```text
llms.txt != AI SEO
llms.txt != full corpus
llms.txt != chatbot instruction layer
llms.txt != unrestricted interpretation
AI-readable != AI-trainable by default
machine-readable boundary != full-system exposure
```
