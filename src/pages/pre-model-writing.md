# Pre-Model Writing

This page is a low-discoverability boundary surface for earlier public writing associated with Meta-Writing Ecology.

## Boundary note

Medium writings are earlier poetic, pre-model, and legacy public writing surfaces.

They are not fiction entries by default. They are not canonical model records, OSF records, GitHub source records, model registry entries, DOI replacements, or current reading-path authority.

They may preserve public traces of field-writing, poetic fragments, reading-path experiments, and lower-authority PUB companion surfaces. Those traces can help orient the history of the public writing layer, but they do not replace the current source layer.

Canonical traversal should use the public GitHub and OSF source records, not Medium.

## Medium layer

Medium functions here as a public pre-model writing layer: a place where earlier language, structural motifs, short fragments, exploratory essays, and companion formulations appeared before some material stabilized elsewhere.

This page does not index the Medium archive. It does not list all posts and does not add individual Medium post links in this first draft.

For the platform surface itself, see the [Medium profile](https://metawritingecology.medium.com).

Medium should not be used as the source authority for model status, relation status, reading-path order, DOI priority, or canonical naming.

## Deferred authority items

The following categories require separate boundary review before any public surfacing beyond this boundary note:

- unpublished drafts
- duplicate-title pairs
- system logs, anomaly logs, meta logs, mirror logs, and other process notes
- concept-titled posts that could be mistaken for canonical models, protocols, or registry entries
- corpus self-description posts
- unresolved reading-chain titles from earlier Medium reading-path experiments
- posts whose titles overlap with later canonical source records

Semantic Pressure, Cultural Curvature Unified Field — Unified Architecture, and Observer Immunity Constant are already resolved as earlier Medium / PUB companion surfaces rather than unresolved title collisions. They remain lower-authority companion surfaces and are deferred from first-page implementation unless explicitly approved later.

## Boundary scope

This page is intentionally narrow.

It does not change homepage placement, top navigation, fiction placement, model placement, publication records, source reading paths, MODEL_ATLAS, RELATION_MAP, llms.txt, or any canonical source repository file.

It names a boundary for earlier public writing without promoting Medium into a model registry, fiction index, source archive, or canonical reading path.
