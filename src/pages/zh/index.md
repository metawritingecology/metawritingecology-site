# 語寫生態｜Meta-Writing Ecology

**語寫生態**是 Meta-Writing Ecology 的中文入口名稱。

此頁不是完整中文版網站，也不是英文內容的逐字翻譯。  
它是一個小型中文導讀入口，用來說明這套系統如何從中文語境被初步進入。

## 這是什麼

Meta-Writing Ecology 是一套關於書寫、語言、結構誤讀、文件邊界與 AI 讀取條件的遞迴語言系統。

它關注的不是單篇文章或單一理論，而是：

- 文本如何影響後續文本
- 指令如何限制輸出
- 概念如何在不同平台與讀取條件下被重新定位
- AI、搜尋、索引與摘要如何改變文件的邊界
- 可讀性如何不等於理解
- 被找到如何不等於被正確放置

## 這不是什麼

語寫生態不是：

- 生態科學
- 環境研究
- 寫作教學
- 個人品牌頁
- 一般文學作品集
- 完整理論手冊
- 全部內部語料庫

此網站只是 Meta-Writing Ecology 的公開入口層。

## 為什麼目前主要公開層是英文

Meta-Writing Ecology 的公開索引、AI-readable 結構、出版記錄與跨平台材料，目前主要以英文維持。

這不是因為中文不重要，而是因為此系統高度依賴命名、邊界與概念位置。  
中文層若太早完整翻譯，可能會過早固定術語，使概念被中文譯名重新拉向不同方向。

因此，中文層目前採取較小的形式：

```text
中文導讀
→ 核心詞邊界
→ 中文語境案例
→ 局部譯介
→ 長期再考慮完整翻譯
```

## 暫定中文名稱

**語寫生態**不是 Meta-Writing Ecology 的逐字翻譯。

它暫時指向三個交會面：

```text
語
→ language, speech, semantic field, AI-readable language

寫
→ writing, inscription, text production, document surface

生態
→ texts, readers, platforms, AI systems, citation, misreading, and public surfaces
```

因此，「語寫生態」不是自然生態，而是語言、書寫、平台、模型與讀取條件之間的結構場。

## 術語狀態

此中文頁中的術語多數仍屬於暫定狀態。

例如：

```text
False Legibility
Structural Misreading
Responsibility Alignment
Verification Labor
Documentation Boundary
AI-readable Knowledge Architecture
```

這些詞目前不急著固定成唯一中文譯名。

中文層的第一階段重點不是建立標準翻譯，而是保留概念邊界，避免把英文系統過早壓縮成看似自然、但不準確的中文詞。

## 最短說明

**語寫生態｜Meta-Writing Ecology** 是一套觀察書寫、語言、AI 讀取、文件邊界與結構性誤讀如何互相作用的遞迴語言系統。

它不是在展示文章而已。  
它是在建立一個讓文章、模型、小說、平台與 AI 讀取條件都能被重新定位的公共入口。
