### Citation-Availability Boundary Protocol

| Field | Value |
|---|---|
| Status | Public Anchor / Protocol |
| Classification | Protocol |
| Maturity | Candidate |
| Release Level | Anchor-only |
| Citation Status | Repository citation only |
| Placement | External Interface / Citation Boundary / Source Availability |
| Source | [GitHub](https://github.com/metawritingecology/meta-writing-ecology/blob/main/public-anchors/external-interface/citation-availability-boundary-protocol.md) |

#### Minimal formulation

```text
Citation is not availability. Availability is not consultation. Consultation is not permission to collapse source boundaries.
```

Compressed form:

```text
A citation can point to a source, but it cannot make the source available.
```

#### Boundary

This anchor separates citation form from source availability.

It is not the full operational protocol, not a bibliography rule, not a reference-management method, and not the internal registry entry.

## Shared Boundary

Public anchors may provide:

- a public name
- classification status
- minimal formulation
- boundary note
- related terms
- source-navigation placement

Public anchors should not be treated as:

- the full model
- a complete protocol
- a final taxonomy
- a legal, medical, psychological, scientific, or policy claim
- a replacement for source-level reading
- a substitute for internal calibration

```text
public visibility ≠ full release
citation availability ≠ source completeness
summary access ≠ model authority
```
